오늘 강의에서는 리눅스 셸과 파일 권한에 대해서 배웠다. 셸 부분에서는
- 셸의 기능
- 셸의 종류
- 셸에서 사용하는  특수문자
	- 정규표현식과는 별개임
- 입출력 리다이렉션
- 환경 설정
	- 셸 변수, 환경 변수
- 환경 설정파일
	- 시스템 환경 설정 파일
	- 사용자 환경 설정 파일

파일 권한에서는
- 파일의 속성
- 접근 권한 변경

등에 대해서 배웠다. 그리고 추가로 정규표현식에 대해서 알아보고 https://regexone.com/ 에서 모든 문제를 풀어보면서 연습을 해봤다.
