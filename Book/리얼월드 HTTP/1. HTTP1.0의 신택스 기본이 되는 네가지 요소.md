# 1. HTTP의 역사
HTTP는 웹 브라우저와 웹 서버가 통신하는 절차와 형식을 규정한 것이다. 웹 브라우저로 웹페이지를 표시할 때 서버로부터 정보를 받아오는 약속이지만, 다양한 서비스의 인터페이스(API)로도 사용되면서 현재 인터넷의 기초가 됐다.
- 1990년: HTTP/0.9
- 1996년: HTTP/1
- 1997년: HTTP/1.1
- 2005년: HTTP/2
최초 버전은 HTML 도큐먼트를 요청해서 가져오기만 하는 단순한 프로토콜이었다.
