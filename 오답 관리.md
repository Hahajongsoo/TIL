# 1. 오답 범주화
1. 개념 부족
2. 문제 이해 못하는 경우(독해 문제)
3. 계산 실수
4. 접근 방법을 모르는 경우
5. 식을 세우는 방법을 모르는 경우
6. 문제를 잘못 보거나 답을 잘 못 체크하는 등 단순 실수

# 2. 오답 관리 방안
- 범주화 한 오답은 교실 내 규칙으로 학생들과 약속
- 학생 오답 하는 과정에서 **항상** 문제 번호 상단에 틀린 이유 번호 작성하도록 지도
	- 안되는 경우 있을 수 있으니 지속적으로 확인
	- 오답 밀리는 경우에 **왜 틀렸는지** 잊을 수 있으니 오답은 밀리지 않는 방향으로 수업
- 틀린 문제들 매쓰플랫 이용하여 유사 문제 제작 후 **매주 금** 클리닉 목적의 숙제
